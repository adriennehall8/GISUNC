Homework 7: Winston-Salem Air Quality Index

Below is an animated map of air quality index data for Winston-Salem, NC during January to February, 2022. I chose this topic because I was interested seeing how air quality was impacted by the Weaver Fertilizer Plant Fire that occurred in late January 2022. This map uses proportional symbols to display the daily AQI for the two sensors in Winston-Salem, NC. This map was created using the QGIS temporal controller. The map was exported as a series of images and then converted into a GIF.


Data used for this project
